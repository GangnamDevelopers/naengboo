# 냉장고를 부탁해(가제) - Backend

## Prerequisites

- Python 3
- Poetry
<!-- - Mongodb -->

## Getting Started

### Install poetry

[check poetry official installation docs](https://python-poetry.org/docs/#installation)

```bash
# install poetry
curl -sSL https://raw.githubusercontent.com/python-poetry/poetry/master/get-poetry.py | python -

# check installation
poetry --version
```

### Run flask

```python
# execute virtualenv
poetry shell

# install dependencies
poetry install

# run flask server
python app.py
```
